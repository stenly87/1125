# Задание: Разработка системы оформления заказ-нарядов (AutoService Order)

**Цель:** Разработать десктопное приложение на C# (**Avalonia UI**, **MySQL**). Приложение позволяет администратору создавать заказ-наряд: выбирать клиента, отмечать выполненные работы из списка, рассчитывать итоговую стоимость со скидкой и сохранять детализацию заказа в базу данных.

## 1. База Данных

**Параметры подключения:**
*   **Сервер:** `192.168.200.13`
*   **БД:** `auto_service_db`
*   **Пользователь:** `student`
*   **Пароль:** `student`

**Таблицы и их назначение**
*   **services** Набор услуг, оказываемых в автосервисе
*   **works** Каждая услуга состоит из нескольких работ, у каждой работы есть стоимость
*   **orders** Заказы клиентов, указывает, какую услугу заказывает клиент и здесь же хранятся
*   **order_items** Список выполненных работ в рамках заказа. Некоторые работы можно пропустить, поэтому итоговая сумма будет зависеть от выполненных работ

## 2. Функциональные требования

### Окно 1: Создание заказа (MainWindow)
*   **Поля:** `TextBox` (Имя клиента), `TextBox` (Авто), `ComboBox` (Услуга из таблицы `services`).
*   **Кнопка:** "Перейти к работам".
*   **Логика:** Валидация полей. Передача `ID услуги` и данных клиента в Окно 2.

### Окно 2: Выполнение работ (WorksWindow)
*   **Отображение:** Список работ из таблицы `works` (фильтр по `service_id`).
*   **Элемент:** Каждая работа — это `CheckBox` + Текст ("Название — Цена").
*   **Логика:**
    *   Пользователь отмечает галочками **только выполненные** работы.
    *   Кнопка "Рассчитать стоимость" собирает список ID отмеченных работ и передает в Окно 3.

### Окно 3: Итоговый чек (ReceiptWindow)
*   **Отображение:**
    *   Список **выполненных** работ (название и цена).
    *   Сумма работ.
    *   Скидка (5% от 5000р, 10% от 10000р).
    *   Итого к оплате.
*   **Сохранение (Важно!):**
    *   При нажатии "Подтвердить" выполняется (по-хорошему в виде **транзакции**):
        1.  `INSERT INTO orders ...` (создается запись заказа, получаем его `NEW ID`).
        2.  Циклом `INSERT INTO order_items (order_id, work_id, price) ...` (сохраняются все отмеченные работы с привязкой к новому ID заказа).
*   **Кнопка:** "Новый заказ" (сброс к Окну 1).
